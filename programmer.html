Game Programmer

Video game programmers are software engineers who create the code that makes the game run. Since current video games are more complex, most programmers focus on one specifc kind of game programming, such as audio or graphics.

Career Requirements

Degree Level Bachelor's degree preferred
Degree Field(s) Information systems, computer science or mathematics
License/Certification Voluntary certifications available
Experience A Portfolio with one or more completed demo projects
Skills Understanding the gaming and programming languages; Strong background in math and science
Job Outlook (2011-2024) 17% growth for progammers
19% growth for applications developers
Mean Annual Salary (2015) $102,160 (for software applications developers)

Other Descriptions


Video game programmers have the option to get a bachelor's degree in either information systems, computer science or mathematics to learn programming languages, operatoin sytems, and advanced math. However, most video game programmers stick with a specialized type of programming, which allows them to focus their skills and to work in a collaborative development process with the whole team.

The types of game programmers are audio engineers, network programmers,graphics programmers and programmers/engineers. Here is a link for more description about the types of video game programming positions.

How it connects to CS?

Programmers use computer science in their work because they have to translate what the designers write into the code. They get a task from the designer or the director to write and decide which code would be the most fun for players and the least taxing on the hardware. And if any bugs appear, they have to first find out what the problem is, then fix it in a way that doesn't break anything else.
