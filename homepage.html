 Game programmers are the people that actually build the game in code. They take the ideas of a designer and translate it into code. They often work in teams to complete and understand a long and big gaming program. If you would like to learn more about becoming a game programmer, click the image to the left.

Creative Game Director

Creative Game Directors oversee all the important decisions for the game. Sometimes, there is only one director for the entire game and other times, there multiple directors for each type of gaming department. They are responsible for the overall creation of the game.
